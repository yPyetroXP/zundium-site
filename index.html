About

Zundium UI Library is a lightweight, modern interface designed by yPyetroXP for Roblox scripting and exploits. Inspired by Fluent UI, it offers a sleek, transparent design with smooth animations and essential features like buttons, toggles, sliders, and more.

How to Use

Load Zundium directly from GitHub into your exploit with this script:

local Zundium = loadstring(game:HttpGet("https://raw.githubusercontent.com/yPyetroXP/zundium/main/src/zundium.lua"))()
local window = Zundium:CreateWindow("Exploit Hub")
window:AddButton("Test", function() print("Clicked!") end)

Press Insert to toggle the UI visibility.

Features

  • Transparent and minimalist design
  • Buttons, toggles, sliders, dropdowns, and textboxes
  • Smooth animations with hover effects
  • Customizable theme
  • Hotkey support (Insert to show/hide)

Documentation

Here’s everything you need to know to use Zundium UI Library effectively in your Roblox exploits.

Creating a Window

Create a new UI window with a custom title:

local Zundium = loadstring(game:HttpGet("https://raw.githubusercontent.com/yPyetroXP/zundium/main/src/zundium.lua"))()
local window = Zundium:CreateWindow("My Exploit")

Adding Elements

Button

Add a clickable button with a callback function:

local button = window:AddButton("Click Me")
button:AddCallback(function()
    print("Button clicked!")
end)

Toggle

Add a toggle switch with an initial state and callback:

local toggle = window:AddToggle("ESP", false)
toggle:AddCallback(function(state)
    print("ESP is now: " .. (state and "ON" or "OFF"))
end)

Slider

Add a slider with min, max, default value, and callback:

local slider = window:AddSlider("Speed", 16, 100, 16)
slider:AddCallback(function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Dropdown

Add a dropdown menu with options and a callback:

local dropdown = window:AddDropdown({"Option 1", "Option 2", "Option 3"})
dropdown:AddCallback(function(selected)
    print("Selected: " .. selected)
end)

Textbox

Add a textbox for user input with a callback on Enter:

local textbox = window:AddTextbox("Enter text")
textbox:AddCallback(function(text)
    print("User entered: " .. text)
end)

Label

Add a simple text label:

local label = window:AddLabel("Welcome to Zundium!")
label:SetText("New text!")

Customizing the Theme

Change the theme colors and transparency:

Zundium:SetTheme({
    PrimaryColor = Color3.fromRGB(20, 20, 25),
    AccentColor = Color3.fromRGB(120, 180, 255),
    TextColor = Color3.fromRGB(240, 240, 240),
    Transparency = 0.6
})

Hotkey

The UI can be toggled with the Insert key by default. Change it with:

Zundium:SetToggleKey("F9") -- Muda para F9

Tips for Exploits

  • Use sliders for fine-tuning values like speed or jump height.
  • Combine toggles with loops for features like ESP or auto-farm.
  • Keep the UI minimized or hidden (Insert) to stay discreet.
